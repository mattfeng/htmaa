# CAD with fusion 360

## useful add-ins
- [Dogbone](https://tapnair.github.io/Dogbone/)

### installing add-ins
- path: `~/Library/Application Support/Autodesk/Autodesk Fusion 360/API/AddIns`

## navigating
- middle mouse drag = pan view
- shift + middle mouse drag = rotate view

## useful keyboard shortcuts
- `S`. shortcuts menu
